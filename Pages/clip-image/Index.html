Clip Image

Learn How to Clip Image



Follow the following steps


First Open Photoshop
Press " Ctrl + N " &
Change the size of Document
Now Click on Ok Button


Open Required Image



Zoom Image
or Focus area


Make Background
to Single Colour



Draw some Shapes
By Using Shape tool


Make Proper Shapes
& Arrange them



Press Ctrl Key & Hold
Select all Shapes

Now Press " Ctrl + E "
To Convert all Shapes
To Single Shape




Make sure shape layer must be
below the Image layer

Press "Alt" key & hold
And Clik on Image layer
Then see the Magic


It's Done now you successfully learn how to Clip Image.
