Background Removeing

Learn How to Remove Background



Follow the following steps


First Open Photoshop
Press " Ctrl + N " &
To open new Document
Now Click on Ok Button

Open Required Images
Or Drag'n Drop in Photoshop


If Image is locked
Go to the Layer Panel
then click on image twice
to unlock it

Now Image will be unlocked




Go to Toolbar
and
Select Eraser tool



Set the size of eraser
By Right Click you can Increase or Decerease the size of eraser

Now start eraseing



You have to be carefull
while eraseing from corners
for that you can reduce the Opacity of eraser

It is good technique
to remove background behind the heair


It's Done now you successfully learn how to Remove Background.
