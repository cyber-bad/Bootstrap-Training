How to Remove Background


Follow the following steps



1. First open the Photoshop application.

2. Now Open Image




3. If Image is locked
Go to the Layer Panel then click on image twice to unlock it
Now Image will be unlocked

4. Go to Toolbar
and Select Eraser tool





5. Set the size of eraser
By Right Click you can Increase or Decerease the size of eraser



6. Now start eraseing





7. You have to be carefull while eraseing from corners
for that you can reduce the Opacity of eraser



8. It is good technique to remove background behind the heair



It's Done now you successfully learn how to remove Background from any Image
